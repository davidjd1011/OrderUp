Este proyecto consiste en el desarrollo e implementación de un Sistema de Gestión de Pedidos (OMS), diseñado para centralizar, automatizar y optimizar todo el ciclo de vida de las órdenes de compra, desde su ingreso hasta su entrega final al cliente.

El objetivo principal es transformar el proceso manual de gestión de órdenes en una operación digital fluida, reduciendo errores, minimizando tiempos de procesamiento y mejorando significativamente la satisfacción del cliente.

Developers: ing. Juan David Villarreall y ing. Frank Josswar Vente
